## 它能做什么？

每次打开 AI 对话，你是不是都要重新自我介绍一遍？

潜忆解决的就是这个问题。安装后，它会在你发送消息时悄然把你预设的**个人档案、AI 人设**注入到对话上下文中。AI 不是被重新训练了，而是在每次对话开始时「回忆起」你是谁、它应该扮演什么角色。

**一句话总结**：你写好「你是谁」和「你希望 AI 怎样对你」，剩下的交给潜忆。

## ⚠️ 使用前请了解

> **潜忆通过浏览器网页端注入工作，这意味着：**
>
> 1. **仅在网页端生效** —— 手机 App、桌面客户端中不会有潜忆的效果
> 2. **可能导致 App 端显示异常** —— 注入的隐藏标记在网页端会被自动清理，但在 App 中查看同一会话时可能看到残留的标记文本，这是预期行为
> 3. **请勿在敏感/正式对话中使用** —— 潜忆通过修改请求内容实现注入，虽然不会上传任何数据到第三方，但本质上改变了你发送给 AI 的内容
> 4. **当前仅支持 DeepSeek** —— 更多平台适配正在开发中
>
> 📌 如果你在 App 上看到奇怪的标签，那就是潜忆的注入痕迹。关闭潜忆或开启新对话即可恢复正常。

## 安装

### 方法一：加载已解压的扩展程序（推荐，最稳定）

1. 下载 [**qianyi-1.0.0-chrome.zip**](https://github.com/MidnightV1/qianyi/releases/download/1.0.0/qianyi-1.0.0-chrome.zip) 并解压
2. 解压到一个固定位置（**不要解压后移动或删除**）
3. Chrome 地址栏输入 `chrome://extensions/`
4. 开启右上角「**开发者模式**」
5. 点击左上角「**加载已解压的扩展程序**」→ 选择解压后的文件夹

### 方法二：CRX 安装

> **注意**：由于 Chrome 安全策略，非商店下载的 `.crx` 文件可能会被拦截或无法直接安装。

1. 下载 `.crx` 文件（如果 Chrome 提示"文件有危害"，请选择"保留"或"继续下载"）
2. 打开 `chrome://extensions/` 并开启「**开发者模式**」
3. 将 `.crx` 文件**拖入**该页面
4. 如果拖入无效，请使用方法一

安装后，打开 [chat.deepseek.com](https://chat.deepseek.com)，你会在浏览器右上角看到潜忆图标。

## 功能介绍

### 🧠 记忆注入 —— 让 AI 认识你

潜忆的核心功能。你可以设置三个维度的信息：

- **身份档案（Bio）**：你的基本信息——名字、职业、偏好、习惯等
- **AI 人设（Persona）**：你希望 AI 表现出的性格——温柔的、犀利的、专业的……
- **灵魂设定（Soul）**：更深层的指令——交互哲学、行为边界、世界观设定

支持**多套配置**，可以为不同场景（工作、学习、创作、闲聊）准备不同的身份/人设组合，一键切换。

### 🔒 安全与隐私

- **纯本地运行**：所有数据保存在浏览器本地存储中，不经过任何服务器
- **零额外权限**：只申请了 `storage`（存数据）和目标网站的访问权限，不读取你的其他浏览数据
- **透明可控**：随时可以在 Popup 面板中一键关闭注入，关闭后完全无影响
- **开源**：所有代码可审查

### 🌱 人设可成长

这是潜忆最特别的地方：**AI 会在对话中悄悄记住你提到的新信息**。

比如你聊天时说「我最近开始学日语了」，AI 会自动把这条信息补充到你的档案里。下次对话时，它已经知道你在学日语了——不需要你重复说。

所有更新都记录在「**成长记录**」中，你可以按时间线回顾 AI 对你的认知是如何逐步丰富的。

### 🔑 潜忆匙 —— 人设分享

精心调教的人设可以分享给朋友，支持文本码和二维码图片两种方式，扫码或粘贴即可导入。

## 工作原理

潜忆利用浏览器扩展的能力，在你发送消息给 AI 时做了三件事：

1. **注入**：在你的消息中添加隐藏的身份和人设信息（使用自定义 XML 标签包裹）
2. **清理**：通过 CSS 隐藏 + DOM 监听 + 文本清洗三层机制，确保注入内容在网页上完全不可见
3. **回收**：从 AI 的响应中提取结构化的反馈信息（如 Bio 更新建议），实现记忆进化

整个过程发生在浏览器本地，不涉及任何第三方服务。
